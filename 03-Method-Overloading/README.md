# Method Overloading

Today I learned Method Overloading in Java.

## Definition
Method Overloading allows multiple methods to have the same name but different parameter lists.

## Key Points
- Same method name
- Different number or type of parameters
- Compile-time Polymorphism

## Example
add(int a, int b)

add(int a, int b, int c)

## Learning Outcome
Learned how Java chooses the appropriate method based on the arguments passed.
