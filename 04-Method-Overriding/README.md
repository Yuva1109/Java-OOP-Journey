# Method Overriding

Today I learned Method Overriding in Java.

## Definition
Method Overriding occurs when a child class provides its own implementation of a method that is already defined in the parent class.

## Key Points
- Same method name
- Same parameters
- Inheritance is required
- Runtime Polymorphism

## Example
Parent class: display()

Child class: display()

## Learning Outcome
Learned how a child class can modify the behavior of a method inherited from its parent class.
